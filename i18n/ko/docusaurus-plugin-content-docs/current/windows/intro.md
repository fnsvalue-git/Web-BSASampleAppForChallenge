# 시작하기

## Guardian Windows (윈도우 로그온)
Guardian Windows Login은 윈도우 OS에서 Guardian-CCS 인증을 이용하기 위해 제공됩니다.   
이 문서는 Guardian Windows Login 사용 방법을 안내 합니다.   

---

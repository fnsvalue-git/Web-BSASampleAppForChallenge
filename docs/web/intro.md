# Get Started

## Web
With the Guardian Web, Guardian-CCS Authentication can be utilized on the web browser.   
This document will describe how to use the Guardian Web Library.

---

## Introduction

Guardian Web Library is made up of [Guardian-JS](https://developers.fnsvalue.co.kr/docs/web/guardianJS/start) which enables direct implementation of authentication and the
[Login-Component](https://developers.fnsvalue.co.kr/docs/web/loginComponent/start) to configure authentication screen in basic setup([Basic Auth]) without further implementation.
